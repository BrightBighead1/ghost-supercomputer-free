# Web Research Skill

## Purpose
Research topics using web search and summarize findings.

## When to Use
- User asks for information on a topic
- User needs comparison or recommendations
- User wants current events or news

## Process
1. Use web search to find relevant sources
2. Read and extract key information
3. Summarize findings concisely
4. Cite sources

## Constraints
- Max 10 search queries per request
- Summarize in under 500 tokens
- Always cite sources
