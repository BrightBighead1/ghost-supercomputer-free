# Data Analysis Skill

## Purpose
Analyze data from Neon database, generate insights, and create summaries.

## When to Use
- User asks for data analysis
- User wants reports or dashboards
- User needs trend identification

## Process
1. Query Neon database using query_database tool
2. Process and summarize results
3. Identify patterns and anomalies
4. Present findings concisely

## Constraints
- Use parameterized queries only
- Keep total DB reads under 1000 rows per query
- Cache frequently accessed data
