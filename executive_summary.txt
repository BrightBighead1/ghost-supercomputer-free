Ghost SuperComputer — Executive Summary

The Ghost SuperComputer is a free, always-on AI agent platform that runs 24/7 without credit cards, student emails, or paid infrastructure. Built on 11 free-tier tools: Suga.app (compute), Neon (database), Supabase (auth), Cloudflare (CDN/storage/vectors), GitAgent (AI agent), n8n (workflows), PocketBase (backup DB), Caddy (proxy), cron-job.org (keep-alive), GitHub (code), and Netdata (monitoring). The system provides a GitAgent-powered AI assistant with git-native memory, multi-model LLM support, declarative tools, and a plugin architecture. Total cost: $0/month forever. Total deploy time: ~75 minutes. Trade-offs include 0.5 GB database (vs unlimited), ~6K vector capacity (vs millions), and 4 containers (vs 24). All data is version-controlled, all limits are documented, and all services have exit strategies.
